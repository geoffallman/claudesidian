# Test Results

This folder contains results from synthetic user testing experiments.

## Structure
- Organize by test date or website/application tested
- Include both raw data and analysis
- Screenshots, recordings, and AI browser logs
- Comparison data between synthetic and real user testing

## File Naming
- Use format: `YYYY-MM-DD_website-name_test-type.md`
- Example: `2025-10-15_ecommerce-checkout_persona-comparison.md`
